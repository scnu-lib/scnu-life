> 华南师范大学校园网址导航

## Description

本网站是模仿[南京大学导航网址](https://github.com/idealclover/Life-in-NJU)做出来的；

主要使用 NextJS 和 TailWind 制作。

## Features

* 可使用三种搜索引擎（百度/必应/谷歌）进行搜索
* 可浏览与华南师范大学相关的网址
* PC端和移动端都可使用
